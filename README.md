# Backgammon Game in C#

## Overview
**Backgammon Game in C#** is a digital implementation of the classic backgammon board game, developed using the **.NET Framework** and **Windows Forms**.  
The project follows an object-oriented design approach and aims to replicate the traditional gameplay experience through a simple and intuitive graphical interface.

---

## Features
- Two-player local gameplay  
- Turn-based system with rule validation  
- Randomized dice rolls using built-in pseudo-random generation  
- Visual representation of the board and pieces using Windows Forms  
- Modular architecture with separate logic and UI layers
